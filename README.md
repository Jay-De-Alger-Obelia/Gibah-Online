# Gibah Online Project

Struktur project ini terdiri dari dua bagian utama:

## 📁 web/
Website Gibah Online (HTML/CSS statis).
- `index.html` - Halaman utama
- `member.html` - Halaman member
- `galeri-lengkap.html` - Halaman galeri lengkap
- `css/` - Semua file stylesheet
- `assets/` - Semua file media (gambar, audio, video)

## 🤖 bot/
Discord bot untuk auto-upload foto ke galeri.
- `bot.js` - File utama bot
- `cloudinary.js` - Integrasi Cloudinary
- `database.js` - Integrasi Supabase
- Jalankan `npm install` lalu `npm start`
- Lihat `SETUP.md` untuk panduan konfigurasi
